## Arsh Origins

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
4. Get a Gemini API key, select a model and start creating teams and delegating tasks

Pacific Mind Teams is an app that helps you delegate tasks to teams of AI agents.

See the Guide in the App for detailed flow and features
