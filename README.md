# Pacific Mind Teams

Pacific Mind Teams is an application designed to help you delegate tasks to dynamic teams of AI agents. 

See the **Guide** inside the app for a detailed overview of the flow and features.

---

## Prerequisites & Setup

Ensure you have **Node.js** installed before proceeding.

1. **Install dependencies:**
```
   npm install
```
2. **Run the app:**
```
npm run dev
```
3. **Configure Environment:**
Set your `GEMINI_API_KEY` in the settings.

Check the guide for more info and attached PRD for developers 


4. **Get Started:**
Obtain a Gemini API key, select your model, and start creating teams to delegate tasks.
